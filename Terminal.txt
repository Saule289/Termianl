pwd  = где находишься  = путь от Диска
cd .. =  выйти на уровень выше
cd ../../  - выйти на нескольког уровней выше
mkdir (название папки) - создать папку
cd - открыть папку

ls  la - показать, что внутри папки с параметрами со скрытымии папаками (столбец)
ls -a - показать, что внутри папки без параметров со скрытымии папаками (в строчку) 
ls - l - показать, что внутри папки c папарметрами без родительской папки (столбец)
ls - показать,что внутри папки без парпаметров без родительской папки (в строчку)

touch - создание файла с расширением (touch text.txt, touch test.docx)
cat >> (название файла) - добавление текста в сам файл, ПОСЛЕ вводе текста ENTER, чтобы ЗАКРЫТЬ ctrl+C 
cat (название текстового файла) = показывает, что внутри файла
open .  = открыть папку в окне (НА МАСе)
cat > название текстового файла  = добавить текстовый файо и сразу открыть его в редактировании для доабления текста в сам файл
cat >НАЗВАНИЕ_ПАПКИ/НАЗВАНИЕ_ФАЙЛа - добавить файл в папку, не заходя в неё, добавить текст в файл

стрелка вверх/вниз - показывает команды, которые вводили раньше в терминале
history  - история всех введенных команд
cd первая буква папки TAB - подставит название подходящнй папки

vim название файла - открыть редактирование в терминале,  i - добавить текст (INSERT),  выйти из режима редактирования Esc,  
выйти из окна редактированич c сохранением изменений :wq ( Enter), :q! - вернуться в терминал без сохранения изменений

mv (название перемещаемого файла) (название папки, куда переносят файл)
mv (название файл) (новое название файла) - переименовать файл


cp (название перемещаемого файла) (название папки, куда переносят файл)

rm (название файла) - удалить ФАЙЛ

rm -r (название папки) - удалить ПАПКУ
